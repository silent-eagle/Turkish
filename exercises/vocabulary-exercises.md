## Упражнения на запоминание

### Упражнение 1: Ассоциации

Создайте ассоциации для запоминания новых слов:

**Пример:**
- **ev** [эв] - дом
  - Ассоциация: "Эв" звучит как "эв-акустика" - дома нужна хорошая акустика

**Задание:** Создайте ассоциации для следующих слов:
1. **kitap** [китап] - книга
2. **su** [су] - вода
3. **güzel** [гюзель] - красивый
4. **öğrenci** [ёренджи] - студент
5. **şehir** [шехир] - город

### Упражнение 2: Группировка слов по темам

Сгруппируйте слова по темам:

**Слова:**
- anne, baba, kardeş, ev, oda, mutfak, yatak, masa, sandalye, kitap, kalem, okul, öğretmen, öğrenci

**Темы:**
- Семья: ________________
- Дом: ________________
- Школа: ________________

**Ответы:**
- Семья: anne, baba, kardeş
- Дом: ev, oda, mutfak, yatak, masa, sandalye
- Школа: kitap, kalem, okul, öğretmen, öğrenci

### Упражнение 3: Словосочетания

Составьте словосочетания с данными словами:

1. **kitap** (книга)
   - новая книга: __________
   - интересная книга: __________
   - читать книгу: __________

2. **ev** (дом)
   - большой дом: __________
   - мой дом: __________
   - идти домой: __________

**Ответы:**
1. yeni kitap, ilginç kitap, kitap okumak
2. büyük ev, benim evim, eve gitmek

## Словосочетания

### Упражнение 1: Глагол + существительное

Составьте словосочетания:

1. **okumak** (читать) + __________ = читать книгу
2. **yemek** (есть) + __________ = есть еду
3. **içmek** (пить) + __________ = пить воду
4. **gitmek** (идти) + __________ = идти в школу
5. **gelmek** (приходить) + __________ = приходить домой

**Ответы:**
1. kitap okumak
2. yemek yemek
3. su içmek
4. okula gitmek
5. eve gelmek

### Упражнение 2: Прилагательное + существительное

Составьте словосочетания:

1. **büyük** (большой) + __________ = большой дом
2. **küçük** (маленький) + __________ = маленькая книга
3. **güzel** (красивый) + __________ = красивый город
4. **yeni** (новый) + __________ = новый студент
5. **eski** (старый) + __________ = старая школа

**Ответы:**
1. büyük ev
2. küçük kitap
3. güzel şehir
4. yeni öğrenci
5. eski okul

### Упражнение 3: Существительное + падеж

Составьте словосочетания с падежами:

1. **ev** (дом) + дательный падеж = __________ (к дому)
2. **kitap** (книга) + винительный падеж = __________ (книгу)
3. **okul** (школа) + местный падеж = __________ (в школе)
4. **şehir** (город) + исходный падеж = __________ (из города)

**Ответы:**
1. eve
2. kitabı
3. okulda
4. şehirden

## Синонимы и антонимы

### Упражнение 1: Синонимы

Найдите синонимы для следующих слов:

1. **güzel** (красивый)
   - a) çirkin
   - b) hoş
   - c) kötü
   - d) eski

2. **büyük** (большой)
   - a) küçük
   - b) geniş
   - c) dar
   - d) yeni

3. **iyi** (хороший)
   - a) kötü
   - b) güzel
   - c) fena
   - d) eski

**Ответы:** 1-b, 2-b, 3-b

### Упражнение 2: Антонимы

Найдите антонимы для следующих слов:

1. **büyük** (большой) → __________
2. **yeni** (новый) → __________
3. **iyi** (хороший) → __________
4. **güzel** (красивый) → __________
5. **uzun** (длинный) → __________

**Ответы:**
1. küçük (маленький)
2. eski (старый)
3. kötü (плохой)
4. çirkin (уродливый)
5. kısa (короткий)

### Упражнение 3: Заполните пропуски синонимами

1. Bu kitap çok __________ (güzel/hoş/iyi)
2. Ev çok __________ (büyük/geniş)
3. Bu yemek çok __________ (kötü/fena)

**Ответы:**
1. güzel/hoş/iyi (все подходят в зависимости от контекста)
2. büyük/geniş
3. kötü/fena

## Игры для запоминания

### Игра 1: "Что это?"

Опишите слово, не называя его. Другие должны угадать.

**Пример:**
- Это место, где мы живем. (ev - дом)
- Это то, что мы читаем. (kitap - книга)

### Игра 2: "Ассоциации"

Один игрок называет слово, другой должен назвать связанное с ним слово.

**Пример:**
- A: okul (школа)
- B: öğrenci (студент)
- A: öğretmen (учитель)
- B: ders (урок)

### Игра 3: "Категории"

Назовите как можно больше слов из определенной категории за минуту.

**Категории:**
- Еда
- Семья
- Школа
- Транспорт
- Цвета

## Рекомендации по изучению лексики

1. **Ежедневное повторение:** Повторяйте новые слова каждый день
2. **Использование в контексте:** Составляйте предложения с новыми словами
3. **Группировка:** Группируйте слова по темам
4. **Визуализация:** Используйте картинки и ассоциации
5. **Практика:** Используйте слова в разговоре и письме
6. **Регулярность:** Изучайте по 10-20 новых слов в день
7. **Повторение:** Повторяйте слова через день, неделю, месяц

## Прогресс изучения лексики

### Месяц 1: ~500-700 слов
- Базовые слова
- Семья
- Дом

- Еда
- Цвета и числа

### Месяц 2: ~1500-2000 слов
- Работа и профессии
- Путешествия
- Здоровье
- Образование
- Разговорные выражения

### Месяц 3: ~2500-3000 слов
- Деловое общение
- Культура и традиции
- Идиомы и сленг
- Специализированная лексика

## Дополнительные упражнения

### Упражнение: Составьте рассказ

Используйте следующие слова в рассказе:
- ev, aile, anne, baba, kardeş, okul, öğrenci, kitap, güzel, büyük

**Пример:**
Benim büyük bir evim var. Evimde ailemle yaşıyorum: annem, babam ve kardeşim. Kardeşim okula gidiyor, o bir öğrenci. Ben kitap okuyorum. Evim çok güzel ve büyük.

### Упражнение: Найдите лишнее слово

В каждой группе найдите лишнее слово:

1. anne, baba, ev, kardeş, çocuk
2. kitap, kalem, okul, masa, öğrenci
3. su, çay, kahve, ekmek, süt

**Ответы:**
1. ev (остальные - члены семьи)
2. masa (остальные связаны со школой/обучением)
3. ekmek (остальные - напитки)
