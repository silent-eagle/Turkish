## Тесты по месяцам

### Тест 1: Месяц 1 - Основы турецкого языка

#### Часть 1: Алфавит и произношение (10 баллов)

1. Какая буква турецкого алфавита произносится как русское "ы"?
   - a) i
   - b) ı
   - c) ü
   - d) ö

2. Буква ğ (yumuşak ge) в слове "dağ" (гора):
   - a) произносится как "г"
   - b) произносится как "х"
   - c) не произносится, удлиняет гласную
   - d) произносится как "й"

3. Выберите правильную транскрипцию слова "şehir":
   - a) [сехир]
   - b) [шехир]
   - c) [чехир]
   - d) [жехир]

**Ответы:** 1-b, 2-c, 3-b

#### Часть 2: Гармония гласных (15 баллов)

4. Выберите правильный суффикс для слова "ev" (дом):
   - a) -da
   - b) -de
   - c) -ta
   - d) -te

5. Выберите правильный суффикс для слова "okul" (школа):
   - a) -de
   - b) -da
   - c) -te
   - d) -ta

6. Какое слово нарушает большую гармонию гласных?
   - a) güzel
   - b) çocuk
   - c) anne
   - d) okul

**Ответы:** 4-b, 5-b, 6-c

#### Часть 3: Времена (25 баллов)

7. Выберите правильную форму глагола "gelmek" (приходить) в настоящем времени для "ben":
   - a) geliyorum
   - b) geliyorsun
   - c) geliyor
   - d) geliyoruz

8. Выберите правильную форму прошедшего времени (di'li geçmiş) для "sen gitmek":
   - a) gittim
   - b) gittin
   - c) gitti
   - d) gittik

9. Выберите правильную форму будущего времени для "biz yapmak":
   - a) yapacağım
   - b) yapacaksın
   - c) yapacak
   - d) yapacağız

10. Какое время используется для действий, о которых узнали от других?
    - a) Di'li geçmiş zaman
    - b) Miş'li geçmiş zaman
    - c) Şimdiki zaman
    - d) Gelecek zaman

**Ответы:** 7-a, 8-b, 9-d, 10-b

#### Часть 4: Падежи (20 баллов)

11. Выберите правильную форму винительного падежа для "kitap" (книга):
    - a) kitap
    - b) kitabı
    - c) kitaba
    - d) kitapta

12. Выберите правильную форму дательного падежа для "ev" (дом):
    - a) ev
    - b) evi
    - c) eve
    - d) evde

13. Выберите правильную форму местного падежа для "okul" (школа):
    - a) okul
    - b) okulu
    - c) okula
    - d) okulda

**Ответы:** 11-b, 12-c, 13-d

#### Часть 5: Притяжательные конструкции (15 баллов)

14. Выберите правильную форму "мой дом":
    - a) benim ev
    - b) benim evim
    - c) ben evim
    - d) ev benim

15. Выберите правильную форму "твоя книга":
    - a) senin kitap
    - b) senin kitabın
    - c) sen kitabın
    - d) kitap senin

**Ответы:** 14-b, 15-b

#### Часть 6: Вопросы и отрицания (15 баллов)

16. Выберите правильную форму вопроса "Ты студент?":
    - a) Sen öğrenci misin?
    - b) Sen öğrenci mi?
    - c) Sen öğrenci değil misin?
    - d) Sen öğrenci değil?

17. Выберите правильную форму отрицания "Я не работаю":
    - a) Ben çalışmıyorum
    - b) Ben çalışmam
    - c) Ben çalışmıyor
    - d) Ben çalışmıyorsun

**Ответы:** 16-a, 17-a

**Итого:** 100 баллов

### Тест 2: Месяц 2 - Развитие навыков общения

#### Часть 1: Причастия и деепричастия (20 баллов)

1. Выберите правильное причастие настоящего времени от глагола "okumak" (читать):
   - a) okuyan
   - b) okuyan
   - c) okumuş
   - d) okuyacak

2. Выберите правильное деепричастие от глагола "gelmek" (приходить):
   - a) gelen
   - b) gelmiş
   - c) gelerek
   - d) gelecek

3. Переведите: "Kitap okuyan öğrenci"
   - a) студент, который читает книгу
   - b) студент, который прочитал книгу
   - c) студент, который будет читать книгу
   - d) студент читает книгу

**Ответы:** 1-a, 2-c, 3-a

#### Часть 2: Условные предложения (20 баллов)

4. Выберите правильную форму условного предложения:
   - a) Eğer hava güzel olursa, dışarı çıkarız
   - b) Eğer hava güzel olur, dışarı çıkarız
   - c) Eğer hava güzel olurdu, dışarı çıkarız
   - d) Eğer hava güzel olursa, dışarı çıktık

5. Выберите правильную форму нереального условия:
   - a) Vaktim olsa, gelirim
   - b) Vaktim olsaydı, gelirdim
   - c) Vaktim olursa, gelirim
   - d) Vaktim olurdu, gelirdim

**Ответы:** 4-a, 5-b

#### Часть 3: Страдательный залог (20 баллов)

6. Выберите правильную форму страдательного залога от "yapmak" (делать):
   - a) yapılmak
   - b) yapılmak
   - c) yapılmak
   - d) yapılmak

7. Переведите: "Bu kitap çok okunuyor"
   - a) Эта книга читает много
   - b) Эта книга читается много
   - c) Эта книга прочитана много
   - d) Эта книга будет читаться много

**Ответы:** 6-a, 7-b

#### Часть 4: Сложные времена (20 баллов)

8. Выберите правильную форму перфекта:
   - a) gelmiştim
   - b) gelmişim
   - c) gelmiş
   - d) geliyordum

9. Какое время используется для действий, завершенных до другого действия в прошлом?
   - a) Miş'li geçmiş
   - b) Di'li geçmiş
   - c) Плюсквамперфект
   - d) Şimdiki zaman

**Ответы:** 8-a, 9-c

#### Часть 5: Модальные конструкции (20 баллов)

10. Выберите правильную форму "Я могу это сделать":
    - a) Bunu yapabilirim
    - b) Bunu yapmalıyım
    - c) Bunu yapmam gerekiyor
    - d) Bunu yapmak istiyorum

11. Выберите правильную форму "Мне нужно уйти":
    - a) Gitmeliyim
    - b) Gitmem gerekiyor
    - c) Gitmek istiyorum
    - d) Gidebilirim

**Ответы:** 10-a, 11-b

**Итого:** 100 баллов

## Ключи к тестам

### Критерии оценивания:

- **90-100 баллов** - Отлично (A)
- **80-89 баллов** - Хорошо (B)
- **70-79 баллов** - Удовлетворительно (C)
- **60-69 баллов** - Достаточно (D)
- **Менее 60 баллов** - Неудовлетворительно (F)

### Рекомендации:

- Если вы набрали менее 70 баллов, рекомендуется повторить соответствующие темы
- Для достижения лучших результатов выполняйте упражнения из соответствующих глав
- Регулярно повторяйте пройденный материал

## Дополнительные материалы

### Тесты для самопроверки

После изучения каждой главы рекомендуется выполнить соответствующие упражнения из глав пособия. Тесты в этом разделе предназначены для комплексной проверки знаний по завершении каждого месяца.

### Как использовать тесты:

1. Выполняйте тесты после завершения соответствующего месяца
2. Не используйте материалы при выполнении теста
3. Проверьте ответы по ключам
4. Проанализируйте ошибки и повторите проблемные темы
5. Повторите тест через несколько дней для закрепления
