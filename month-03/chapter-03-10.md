# Глава 3.10: Составление резюме и официальных документов

## Введение

Для учебы и работы нужен грамотный CV, сопроводительное письмо и умение заполнять официальные формы. В этой главе — структура резюме, типовые формулы и примерные шаблоны.

**Цели главы:**
- Освоить структуру CV (резюме) на турецком
- Научиться писать краткое сопроводительное письмо
- Заполнять официальные формы и заявления
- Закрепить через упражнения и мини-шаблоны

---

## Теоретическая часть

### 1. Структура CV (kısa)
- Контакты: имя, телефон, email, город.  
- Цель/профиль (opsiyonально): 1–2 предложения.  
- Опыт (İş deneyimi): компания, должность, даты, обязанности, достижения.  
- Образование (Eğitim): вуз, факультет, годы.  
- Навыки (Yetenekler): языки, софт, сертификаты.  
- Дополнительно: проекты, волонтерство, рекомендации.

### 2. Формулы для опыта и достижений
- **Sorumluluklar:** görev tanımı.  
- **Başarılar:** ölçимые результаты: **%X iyileştirme**, **maliyet düşürme**, **süre kısaltma**.  
- Глаголы: **yönetmek**, **geliştirmek**, **optimize etmek**, **koordin etmek**, **raporlamak**, **destek olmak**.

### 3. Сопроводительное письмо (önyazı)
- Обращение: **Sayın ...**, **Merhaba** (если полуформально).  
- 1 абзац: кто вы, позиция/цель.  
- 2 абзац: почему вы подходите, ключевые навыки.  
- 3 абзац: готовность к встрече, благодарность.  
- Завершение: **Saygılarımla**, **İyi çalışmalar dilerim**.

### 4. Официальные формы и заявления
- Частые поля: **Ad Soyad**, **Adres**, **Doğum Tarihi**, **TC Kimlik No**, **İmza**, **Tarih**.  
- Формулы просьбы: **Bilgilerinize arz ederim**, **Gereğini rica ederim**.

## Примеры

**Мини-CV (фрагмент)**  
```
Ad Soyad: Elif Yılmaz
E-posta: elifyilmaz@mail.com | Telefon: +90 5xx xxx xx xx | İstanbul
Profil: Veri analizi ve raporlama alanında 3 yıllık deneyim.
İş Deneyimi:
- Veri Analisti, ABC Şirketi (2021- ) – Raporlama, dashboard geliştirme, maliyet analizi.
Eğitim: Boğaziçi Üniversitesi, İşletme (2017-2021)
Yetenekler: Excel, SQL, Python; İngilizce B2, Türkçe C1
```

**Сопроводительное (кратко)**  
```
Sayın Yetkili,
İlan ettiğiniz Veri Analisti pozisyonu için başvurmak isterim.
3 yıllık deneyimim ve raporlama becerilerimle ekibinize katkı sağlayacağıma inanıyorum.
Müsait olduğunuzda görüşmekten memnuniyet duyarım.
Saygılarımla,
Elif Yılmaz
```

## Упражнения

### 1. CV-блоки
Заполните шаблон: контакты, опыт (1 позиция), образование, навыки.

### 2. Сопроводительное письмо
Напишите 6–8 предложений по структуре (обращение, 3 абзаца, завершение).

### 3. Формы/заявления
Переведите на турецкий: имя, адрес, дата рождения, подпись, дата; добавьте формулу вежливой просьбы.

### 4. Лексика достижений
Составьте 3 пункта с измеримыми результатами (проценты, сроки, стоимость).

## Словарь главы (≈50 единиц)
- **özgeçmiş / CV** — резюме  
- **önyazı** — сопроводительное письмо  
- **iş deneyimi** — опыт работы  
- **görev tanımı** — описание обязанностей  
- **başarı** — достижение  
- **sorumluluk** — ответственность  
- **eğitim** — образование  
- **yetenek** — навык  
- **sertifika** — сертификат  
- **referans** — рекомендация  
- **ilan** — вакансия/объявление  
- **başvuru** — заявка  
- **mülakat** — интервью  
- **görüşme** — встреча  
- **uygunluk** — соответствие  
- **başvurmak** — подаваться  
- **katkı sağlamak** — вносить вклад  
- **geliştirmek** — развивать  
- **yönetmek** — управлять  
- **analiz etmek** — анализировать  
- **optimize etmek** — оптимизировать  
- **raporlamak** — отчитываться  
- **sunmak** — представлять  
- **arz ederim** — с уважением представляю (формула)  
- **gereğini rica ederim** — прошу сделать необходимое
