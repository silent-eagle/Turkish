# Онлайн-ресурсы

В этом разделе собраны полезные онлайн-ресурсы для изучения турецкого языка.

---

## Образовательные сайты

### Комплексные платформы

1. **Duolingo**
   - URL: https://www.duolingo.com
   - Описание: Интерактивное изучение языка через игры
   - Особенности: Бесплатно, мобильное приложение, ежедневные задания
   - Уровень: Начальный-Средний

2. **Babbel**
   - URL: https://www.babbel.com
   - Описание: Структурированные курсы турецкого языка
   - Особенности: Платно, интерактивные уроки, произношение
   - Уровень: Начальный-Продвинутый

3. **Busuu**
   - URL: https://www.busuu.com
   - Описание: Социальная сеть для изучения языков
   - Особенности: Общение с носителями, сертификаты
   - Уровень: Начальный-Продвинутый

4. **Memrise**
   - URL: https://www.memrise.com
   - Описание: Изучение через запоминание слов
   - Особенности: Мнемотехники, видео носителей
   - Уровень: Начальный-Средний

### Специализированные сайты

1. **TurkishClass101**
   - URL: https://www.turkishclass101.com
   - Описание: Аудио и видео уроки турецкого
   - Особенности: Подкасты, транскрипции, словари
   - Уровень: Начальный-Продвинутый

2. **Learn Turkish**
   - URL: https://learnturkish.com
   - Описание: Грамматика и упражнения
   - Особенности: Бесплатные уроки, тесты
   - Уровень: Начальный-Средний

3. **Turkish Grammar**
   - URL: https://turkishgrammar.com
   - Описание: Подробный грамматический справочник
   - Особенности: Таблицы, примеры, упражнения
   - Уровень: Все уровни
