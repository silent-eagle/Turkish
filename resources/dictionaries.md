# Словари и справочники

В этом разделе собраны полезные словари и справочники для изучения турецкого языка.

---

## Онлайн-словари

### Турецко-русские словари

1. **Tureng**
   - URL: https://tureng.com
   - Описание: Один из лучших турецко-английских и турецко-русских словарей
   - Особенности: Примеры использования, произношение, синонимы

2. **Sözlük**
   - URL: https://sozluk.gov.tr
   - Описание: Официальный словарь турецкого языка (Türk Dil Kurumu)
   - Особенности: Толковый словарь, этимология, примеры

3. **Reverso Context**
   - URL: https://context.reverso.net
   - Описание: Словарь с контекстными примерами
   - Особенности: Примеры из реальных текстов, переводы предложений

4. **Glosbe**
   - URL: https://glosbe.com
   - Описание: Многоязычный словарь с примерами
   - Особенности: Большая база примеров, произношение

### Толковые словари (только турецкий)

1. **Türk Dil Kurumu (TDK) Sözlük**
   - URL: https://sozluk.gov.tr
   - Описание: Официальный толковый словарь турецкого языка
   - Особенности: Авторитетный источник, подробные определения

2. **Nisanyan Sözlük**
   - URL: https://www.nisanyansozluk.com
   - Описание: Этимологический словарь турецкого языка
   - Особенности: История слов, происхождение

## Рекомендации по использованию словарей

### Выбор словаря

1. **Для начинающих:**
   - Используйте турецко-русские словари с транскрипцией
   - Обращайте внимание на примеры использования

2. **Для среднего уровня:**
   - Комбинируйте турецко-русские и толковые словари
   - Изучайте синонимы и антонимы

3. **Для продвинутого уровня:**
   - Используйте толковые словари на турецком
   - Изучайте этимологию слов

### Эффективное использование

1. **Не переводите дословно:**
   - Учитывайте контекст
   - Изучайте примеры использования

2. **Проверяйте несколько источников:**
   - Сравнивайте определения
   - Изучайте синонимы

3. **Ведите свой словарь:**
   - Записывайте новые слова
   - Добавляйте примеры
   - Отмечайте частоту использования
